Loan Default Prediction System (LDaaS)

A Loan Default Prediction System that leverages Excel, Python, Machine Learning algorithms, and Tableau to analyze borrower data and predict the probability of loan default. The solution helps financial institutions identify high-risk borrowers, reduce NPAs, and make data-driven lending decisions.

Key Features

  • Data preprocessing & feature engineering (income, DTI, credit history, etc.)
  • Model training using Logistic Regression, Random Forest, Decision Tree, XGBoost, and SVM
  • Model evaluation using accuracy, precision, recall, F1-score, ROC-AUC
  • Interactive Tableau dashboards for trend analysis & borrower risk profiling
  • Final outcome: Best-performing ML model + Business Insights
